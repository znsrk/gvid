# gvid Plus

An educational platform for learning and studying with interactive tools. AI creates entire courses and learning material from the stuff provided. 

## Tech Stack

- **Frontend:** React + Vite
- **Styling:** CSS with dynamic theming
- **Architecture:** Modular plugin-based system

## Features

- **Plugin System** — Extensible architecture allowing custom plugins to enhance functionality
- **Theme Customization** — Switch between color themes to personalize your experience
